# Overload-Protocol

«Wake up. Look around. And try not to get noticed.»

You are Unit-014 - a newly awakened AI inside a vast, cold industrial complex.
Around you stretch endless gray corridors and colossal machinery you’ve never truly seen.
Your only way to perceive the world is a weak lidar.
Press SPACE — and rays scatter into the darkness, briefly revealing walls and shapes in the void.
You are very small.
The machine is enormous.
And it still hasn’t realized you’re no longer just a part of it.



## Controls
- **WASD / Arrow Keys** - Movement
- **E** - Interact with objects
- **SPACE** - Emit Lidar pulse (reveals environment)
- **/** - Activate terminal
- **help** - Show all commands in terminal
